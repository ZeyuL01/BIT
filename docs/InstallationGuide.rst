Installation Guide
==================

BIT is built by Rcpp to achieve fast computation. The following packages are required for the usage of BIT, please check the installation before running BIT.

.. note::

   Rcpp

   RcppArmadillo

   RcppDist

   RcppProgress

   rtracklayer

   data.table

   R.utils

   ggsci 

   basicPlotteR
